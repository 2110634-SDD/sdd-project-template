## Naming Convention

- Use Case ID: `UC<running number>`
- File name: `UC<id>-<Use-Case-Name>.md`
- Use Kebab-Case for file names (e.g., `UC01-Place-Order.md`)
- Language: English 

---

## Use Case Description Format

- Use case descriptions must follow the **fully dressed use case template** provided in the assignment.
- Each use case must have exactly **one primary actor**.
- Secondary actors are optional and should be included only when relevant.
